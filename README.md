
## The Workflow

This is a collection of experimental, fast-prototyped projects built using AI-assisted tools. These projects leverage modern AI-assisted workflows:

1. Spec Documents (PRD + Requirements)
2. System Architecture & Tech Design
3. Testcase Specifications
4. Project Setup & Detailed Design Plan
5. Agent Execution : Built using tools like Cursor, OpenCode or Cline.


## Navigation & Setup

Each project in this repository is completely self-contained.

1. **Browse**: Click any project link in the catalog above to open its folder.
2. **Read**: Check the individual `README.md` inside that folder for specific prerequisites and features.
3. **Run**: Navigate into the folder via your terminal to install dependencies and execute:
    
    bash
    
    ```
    cd project-folder-name
    ```
